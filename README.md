# Demographic Data Analyzer

This project analyzes demographic census data using **Python** and **Pandas**.
It is part of the **FreeCodeCamp Data Analysis with Python Certification**.

The program reads a dataset and extracts useful insights about education, income, work hours, and countries.

---

##  Dataset

The dataset used is **adult.data.csv**, which contains demographic information such as:

* Age
* Education level
* Occupation
* Race
* Gender
* Hours worked per week
* Native country
* Salary category (>50K or <=50K)

Each row represents information about one individual.

---

##  Features / Analysis Performed

The program performs the following analysis:

1. Count the number of people in each race category
2. Calculate the average age of men
3. Find the percentage of people with a Bachelor's degree
4. Determine the percentage of people with higher education earning more than 50K
5. Determine the percentage of people without higher education earning more than 50K
6. Find the minimum number of work hours per week
7. Calculate the percentage of people earning >50K among those who work the minimum hours
8. Identify the country with the highest percentage of people earning >50K
9. Find the most common occupation in India among people earning >50K
